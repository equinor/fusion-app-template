
## Getting started

1. Update the `README.md`
2. _[optional]_ setup [changeset](https://github.com/changesets/changesets/blob/main/docs/intro-to-using-changesets.md)
    - utility for keeping track of changes in the repo and maintain correct semantic version
3. create a `SECURITY.md` file
4. create a `LICENSE` file
